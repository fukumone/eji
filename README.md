Eji
====================
Provide convenient access for emoji

## dependencies

```
$ go get github.com/urfave/cli
```

## install

```
$ go install github.com/fukumone/eji
```

## Usage

### Find emoji from keywords

```
$ eji :smile:
# output ðŸ˜„
```

### Find keyword from emoji

```
$ eji ðŸ˜„
# output :smile:
```

### Display a list of emoji

```
$ eji index
# or eji i
```
